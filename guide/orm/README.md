# DBX ORM

An <abbr title="Object-Relational Mapping">ORM</abbr> comes with DBX.

To import DBX ORM:

```crystal
require "dbx"
require "dbx/adapter/pg" # or require "dbx/adapter/sqlite"
require "dbx/orm"
```
